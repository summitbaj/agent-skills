\# Agent Skills



A collection of AI agent skills installable via \[skills.sh](https://skills.sh).



\[!\[skills.sh](https://skills.sh/b/summitbaj/agent-skills)](https://skills.sh/summitbaj/agent-skills)



\## Install all skills



\\```bash

npx skills add summitbaj/agent-skills

\\```



\## Install a specific skill



\\```bash

npx skills add summitbaj/agent-skills --skill powerapps-canvas-yaml-generator

\\```



\## Skills



| Skill | Description |

|---|---|

| \[powerapps-canvas-yaml-generator](./skills/powerapps-canvas-yaml-generator/) | Generates valid Power Apps Source Code YAML for canvas app screens |
